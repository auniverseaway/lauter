Introduction

Lauter is a lightweight grid system.
It's designed with the following tenets:

  • Responsive
  • Lightweight
  • Easy to use
  • Based on Flexbox
  • All cells add up to 100% (or not)
  • Support near-arbitrary column widths
  • Support vertical alignment

Breakpoints

Lauter matches common responsive breakpoints.

  • Mobile 0-543 PX
  • Small 544-767 PX
  • Medium 768-991 PX
  • Large 992-1199 PX
  • Extra 1200 PX+

Cell Widths

Overview

Lauter does not mandate a 12 / 16 / 24 column grid. You can specify Cell widths in 5% increments. They can add up to 100%, or you can leave out widths to take up remaining space in the grid. You can also choose to not fill the entire area of a grid as shown in the last example.

Basic example

You can declare sizes on a per-viewport basis. Below is a sample that sets the left column to 30% for medium and above viewports. Widths are opt-in, you don't have to specify a width for all Cell elements. In this case, the right column does not have a width specified. It simply takes up the remainder of the available space.
